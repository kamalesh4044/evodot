# Evodot FPS

A multiplayer First-Person Shooter game built in Godot 4.

## Features
- Full Multiplayer Support (Host/Join)
- First-Person and Third-Person Perspectives
- Dynamic mixamo animations syncing across network
- Fully animated characters reacting to movement states (Idle, Run, Jump, Shoot)
- Three distinct weapons (Assault Rifle, SMG, Shotgun)
- Sliding mechanics and advanced physics

## Getting Started
1. Open the project folder in Godot 4.
2. Open `Main.tscn` or `Lobby.tscn`.
3. Press Play to start the game.
4. You can host a game and have another instance join via `127.0.0.1`.

## Controls
- **W, A, S, D**: Move
- **Space**: Jump
- **Ctrl / C**: Slide
- **Left Click**: Fire Weapon
- **Right Click**: Aim Down Sights (ADS)
- **1, 2, 3**: Switch Weapons
- **R**: Reload
- **Esc**: Free mouse cursor
